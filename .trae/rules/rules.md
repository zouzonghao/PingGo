1、本地为Windows。
2、终端为powershell，如果要执行终端命令，需要使用powershell支持的命令。
3、使用中文对话。
4、使用go run main.go 启动项目，不需要构建。
5、本项目文件夹为web服务监控程序的go版本实现，效果为形成一个静态编译的go程序，方便部署。
6、简化通知功能，只实现邮件通知，通过resend-api发送通知邮件。
7、使用 Go 标准库 net/http 或 Web 框架（如 Gin, Echo）提供静态文件服务和 API 路由的统一后端。
8、项目构建需要将前端静态文件编译到go程序中。
9、我的项目用户为中国用户，界面语言为中文。
10、除非必要，不要引入太多JS和css文件，否则会影响加载速度。比如使用系统提供的字体和图标库。
